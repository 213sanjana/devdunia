Common Examples

Simple text:

"Hello World"

→ "JxF12TrwUP45BMd"

Short text:

"test"

→ "3yZe7d"

JSON data:

{"name": "John", "age": 30}

→ "7vxA8pHnqW5aXc2Y9zB"

Binary data:

[0x48, 0x65, 0x6C, 0x6C, 0x6F]

→ "StV1DL6CwTryKyV"
