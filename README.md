## 🚀 Getting Started

### Use Online (Recommended)
Visit [devdunia.com](https://devdunia.com) and start using tools immediately!

### Run Locally
```bash
git clone https://github.com/echobash/devdunia.git
cd devdunia
python -m http.server 8000  # or npx serve .
```
