YAML ⇄ JSON Converter

Convert between YAML and JSON formats with proper syntax and formatting.

Conversion Examples

YAML Object:

name: John Doe
age: 30
active: true
skills:
  - Python
  - JavaScript

→ JSON equivalent

JSON Object:

{
  "name": "John Doe",
  "age": 30,
  "active": true,
  "skills": ["Python", "JavaScript"]
}

→ YAML equivalent
