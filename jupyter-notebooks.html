Quick Reference

Keyboard Shortcuts:

  • • Shift + Enter - Run cell
  • • Ctrl + Enter - Run cell (stay)
  • • A - Insert cell above
  • • B - Insert cell below
  • • DD - Delete cell
  • • M - Markdown mode
  • • Y - Code mode

Useful Extensions:

  • • jupyter_contrib_nbextensions
  • • jupyterlab-git - Git integration
  • • jupyterlab-lsp - Language server
  • • jupyterlab-drawio - Diagrams
  • • jupyterlab-toc - Table of contents
