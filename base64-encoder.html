Base64 Encoder

Convert text and binary data to Base64 encoding format.

Original Size: 0 B
Base64 Size: 0 B
Overhead: 0%
Padding: 0 chars

Common Examples

Simple text:

"Hello World"

→ "SGVsbG8gV29ybGQ="

Binary data:

[0x48, 0x65, 0x6C, 0x6C, 0x6F]

→ "SGVsbG8="

JSON data:

{"name": "John", "age": 30}

→ "eyJuYW1lIjoiSm9obiIsImFnZSI6MzB9"

File content:

Image files, PDFs, etc.

→ Long Base64 string
